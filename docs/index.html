개요 기술 스택 페이지 구성 데이터베이스 API 관리자 주요 기능 설정 가이드

프로젝트 개요

아빠트리(APPATREE)는 시니어(고령자)를 위한 음성 기반 오디오북 검색 및 재생 웹 애플리케이션입니다.

음성 검색

Web Speech API 또는 Whisper STT로 한국어 음성 인식. 말로 오디오북을 찾을 수 있습니다.

YouTube 연동

YouTube Data API v3로 오디오북 검색, 재생. 인메모리 캐시로 API 쿼터 보호.

AI 추천

Gemini 2.5 Flash LLM 기반 맞춤 오디오북 추천. 대화형 인터페이스.

관리자 대시보드

카테고리, 큐레이션, AI/YouTube/음성 설정, 공지사항, 분석 차트.

이어 듣기

재생 위치 자동 저장. 나중에 이어서 들을 수 있습니다.

시니어 UI

큰 글씨, 큰 버튼, 고대비 모드, TTS 음성 안내, 수면 타이머.

설정 가이드

1. Supabase 설정

  1. supabase.com 에서 프로젝트 생성
  2. Settings → Database → Connection string 복사
  3. .env.local에 DATABASE_URL=postgresql://... 추가

2. 환경변수

DATABASE_URL=postgresql://postgres:password@host:5432/postgres
JWT_SECRET=your-jwt-secret
VITE_APP_ID=your-app-id
OAUTH_SERVER_URL=https://oauth.example.com
YOUTUBE_API_KEY=AIzaSy...
BUILT_IN_FORGE_API_URL=https://forge.example.com
BUILT_IN_FORGE_API_KEY=your-forge-key

3. 실행

pnpm install          # 의존성 설치
pnpm run db:push      # DB 마이그레이션
pnpm run dev          # 개발 서버 (http://localhost:3000)

4. 배포 (Vercel)

  1. GitHub에 Push → Vercel 자동 배포
  2. Vercel 대시보드에서 환경변수 설정
  3. Build Command: vite build
  4. Output Directory: dist/public
