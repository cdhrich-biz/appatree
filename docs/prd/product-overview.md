# 제품 개요

## 1. 제품명

**APPATREE (아빠트리)** - 시니어 오디오북

## 2. 한줄 요약

시력이 불편한 시니어 사용자를 위한 음성 기반 오디오북 검색 및 재생 모바일 웹앱

## 3. 제품 목표

- 시니어 사용자가 음성만으로 오디오북을 검색하고 재생할 수 있는 환경 제공
- 3탭 이내에 모든 핵심 기능에 도달할 수 있는 단순한 UX
- AI 추천으로 사용자 맞춤형 오디오북 발견 지원
- 오프라인에서도 기본 기능 사용 가능한 PWA

## 4. 타겟 사용자

### 주요 사용자
- **연령**: 60세 이상 시니어
- **특성**: 시력 저하, 스마트폰 사용에 익숙하지 않음
- **니즈**: 크고 선명한 UI, 음성 우선 인터랙션, 단순한 네비게이션
- **디바이스**: 스마트폰 (Android Chrome, Samsung Internet, iOS Safari)

### 사용 시나리오
1. 시니어 사용자가 음성 버튼을 눌러 "역사 소설 추천해줘"라고 말함
2. 앱이 음성을 인식하여 관련 오디오북 검색 결과를 보여줌
3. 원하는 오디오북을 탭하면 바로 재생 시작
4. AI 채팅에서 "편안한 마음으로 들을 수 있는 책은?"이라고 질문하면 맞춤 추천

## 5. 핵심 가치

| 가치 | 설명 |
|------|------|
| **음성 우선** | 타이핑 없이 음성만으로 검색/명령 가능 |
| **고접근성** | WCAG 2.2 AA 준수, 16.5:1 이상 명암비 |
| **단순함** | 3탭 원칙, 최소한의 메뉴 구조 |
| **AI 추천** | OpenAI 기반 맞춤형 오디오북 추천 |
| **오프라인** | PWA로 설치 가능, 기본 기능 오프라인 지원 |

## 6. 비즈니스 목표

- 시니어 사용자의 오디오북 접근성 향상
- 음성 인터랙션 기반 차별화된 UX 제공
- PWA로 앱스토어 없이 빠른 배포 및 업데이트

## 7. 성공 지표

| 지표 | 목표 |
|------|------|
| 음성 인식 성공률 | 85% 이상 |
| 첫 화면 로딩 (FCP) | < 2초 (3G) |
| 음성→검색 결과 | < 3초 |
| AI 채팅 응답 | < 5초 (스트리밍) |
| 접근성 명암비 | 16.5:1 이상 |
