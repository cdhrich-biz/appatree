# 시니어 UX 디자인 시스템

## 1. 디자인 원칙

| 원칙 | 설명 |
|------|------|
| **음성 우선** | 타이핑보다 음성 입력을 기본으로 |
| **3탭 원칙** | 모든 기능 3탭 이내 도달 |
| **명확한 피드백** | 시각 + 음성 동시 피드백 |
| **실수 용서** | 되돌리기, 확인 단계 제공 |
| **고대비** | WCAG AAA 수준 명암비 |

## 2. 타이포그래피

```css
/* 시니어 타이포그래피 스케일 */
--text-senior-body:    18px;   /* 본문 최소 크기 */
--text-senior-button:  20px;   /* 버튼 텍스트 */
--text-senior-heading: 24px;   /* 소제목 */
--text-senior-title:   32px;   /* 대제목 */
--line-height:         1.6~1.8; /* 줄 간격 */
```

### 설정별 크기 조절
- **작게**: 본문 18px
- **보통**: 본문 20px (기본)
- **크게**: 본문 24px

## 3. 색상 팔레트

### 주요 색상

| 용도 | 색상 | HEX | 비고 |
|------|------|-----|------|
| Primary (액션) | 진한 초록 | `#1B5E20` | 테마 색상, 메인 CTA |
| Primary Hover | 더 진한 초록 | `#145218` | 호버/포커스 |
| Background | 흰색 | `#FFFFFF` | 기본 배경 |
| Text Primary | 짙은 회색 | `#1F2937` | 본문 텍스트 |
| Text Secondary | 중간 회색 | `#4B5563` | 보조 텍스트 |
| Border | 연한 회색 | `#E5E7EB` | 경계선 |
| Card BG | 매우 연한 회색 | `#F9FAFB` | 카드 배경 |
| Danger | 빨강 | `#DC2626` | 삭제/오류 |

### 명암비
- 본문 텍스트: **16.5:1** (WCAG AAA 초과)
- 버튼 텍스트: **최소 4.5:1** (WCAG AA)
- 색상에만 의존하지 않음 (아이콘 + 텍스트 함께 사용)

## 4. 인터랙션 요소

### 터치 타겟
- **최소 크기**: 48px x 48px
- **버튼 높이**: 56px+ (대형 액션 버튼)
- **음성 버튼**: 170px 원형
- **탭 바 아이콘**: 48px

### 버튼 스타일

```
[Primary]   bg-green-700, text-white, rounded-2xl, min-h-56px
[Secondary] bg-gray-100, text-gray-800, rounded-xl
[Ghost]     bg-transparent, hover:bg-gray-100
[Danger]    text-red-600, hover:bg-red-50
```

### 상태 피드백
- **호버**: 배경색 변화 (200ms transition)
- **포커스**: 2px ring, ring-green-500
- **활성**: scale(0.98) 트랜스폼
- **비활성**: opacity-50, cursor-not-allowed

## 6. 접근성 (WCAG 2.2 AA)

### 적용 항목
- [x] 모든 인터랙티브 요소 `aria-label` 부여
- [x] 키보드 내비게이션 (Tab, Enter, Escape)
- [x] 포커스 가시성 (2px ring)
- [x] 스크린 리더 호환 (`role`, `aria-live`)
- [x] 색상 대비 4.5:1 이상
- [x] 텍스트 크기 조절 가능 (설정)
- [x] 모션 줄이기 (`prefers-reduced-motion` 대응)

### 음성 접근성
- STT: 한국어(ko-KR) 음성 인식
- TTS: 검색 결과, AI 응답 음성 출력
- 속도 조절: 0.75x ~ 1.25x
