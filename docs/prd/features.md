# 기능 명세

## 페이지 구조

```
/                → 홈 (음성 검색 + 카테고리)
/search          → 검색 결과
/player          → 오디오북 플레이어
/chat            → AI 채팅 추천
/library         → 내 서재 (즐겨찾기 + 최근 재생)
/settings        → 설정
```

---

## 1. 홈 화면 (`/`)

**파일:** `client/src/pages/Home.tsx`

### 구성 요소
- **상단 바**: 로고(🎧) + "아빠트리" 타이틀 + 설정 아이콘
- **음성 버튼**: 170px 원형, 초록색, 펄스 애니메이션
- **상태 메시지**: "듣고 있습니다", "검색 중입니다", "다시 말씀해주세요"
- **카테고리 그리드**: 2열 6개 버튼
  - 📖 소설 | ✍️ 에세이
  - 🏛️ 역사 | 💼 경제
  - 🌱 자기계발 | ⭐ 인기 오디오북
- **하단 탭 바**: 홈 | 검색 | AI 채팅 | 내 서재

### 동작
1. 음성 버튼 탭 → 마이크 권한 요청 → 녹음 시작
2. 음성 인식 완료 → 자동으로 `/search?q={인식결과}` 이동
3. 카테고리 버튼 탭 → `/search?q={카테고리명}` 이동
4. TTS로 상태 변경 시 음성 피드백 제공

---

## 2. 검색 결과 (`/search`)

**파일:** `client/src/pages/SearchResults.tsx`

### 구성 요소
- **헤더**: 뒤로가기 + 검색어 표시
- **검색 입력**: 텍스트 입력 + 음성 버튼
- **정렬 옵션**: 관련성 | 최신순 | 조회수
- **결과 카드**: 썸네일 + 제목 + 채널명 + 재생시간 + 조회수 + 재생 버튼

### 동작
1. 검색어로 YouTube API 호출 (현재 목업 데이터)
2. 정렬 버튼으로 결과 재정렬
3. 결과 카드 탭 → `/player?id={videoId}&title={제목}` 이동
4. 결과 없으면 "AI에게 추천받기" 버튼 표시

---

## 3. 플레이어 (`/player`)

**파일:** `client/src/pages/Player.tsx`

### 구성 요소
- **영상 영역**: YouTube IFrame (16:9 비율)
- **정보 영역**: 제목 + 채널명
- **프로그레스 바**: 48px 높이, 드래그 가능, 현재/전체 시간 표시
- **컨트롤 바**:
  - 30초 되감기 | 재생/일시정지 | 30초 건너뛰기
  - 재생 속도: 0.75x(느림) | 1.0x(표준) | 1.25x(빠름)
  - 볼륨 슬라이더

### 동작
1. URL 파라미터에서 videoId 추출 → YouTube IFrame 로드
2. 재생/일시정지 토글
3. 30초 단위 이동
4. 재생 속도 변경 (시니어: 0.75x 기본 권장)

---

## 4. AI 채팅 (`/chat`)

**파일:** `client/src/pages/AIChat.tsx`

### 구성 요소
- **헤더**: 뒤로가기 + "AI 채팅" + "오디오북 추천 상담"
- **메시지 영역**: 버블형 UI (사용자: 초록, AI: 회색)
- **프리셋 메시지**: 4개 빠른 질문 버튼
  - "오늘 듣기 좋은 책 추천해주세요"
  - "감동적인 이야기를 찾고 있어요"
  - "역사 관련 책을 추천해주세요"
  - "편안한 마음으로 들을 수 있는 책은?"
- **입력 영역**: 텍스트 입력 + 음성 버튼 + 전송 버튼

### 동작
1. 프리셋 또는 직접 입력으로 질문
2. OpenAI API로 추천 요청 (현재 시뮬레이션)
3. AI 응답에 TTS 재생 버튼 제공
4. 추천 오디오북 탭 → 플레이어로 이동

## 5. 내 서재 (`/library`)

**파일:** `client/src/pages/Library.tsx`

### 구성 요소
- **헤더**: 뒤로가기 + "내 서재"
- **탭**: 즐겨찾기 | 최근 재생
- **도서 카드**: 썸네일 + 제목 + 채널명 + 마지막 재생일
  - 즐겨찾기: 재생 버튼 + 삭제 버튼
  - 최근 재생: 재생 버튼만

### 동작
1. localStorage에서 즐겨찾기/최근 재생 목록 로드
2. 재생 버튼 → 플레이어로 이동
3. 삭제 버튼 → 즐겨찾기에서 제거
4. 빈 목록 시 "책 찾아보기" 안내

---

## 6. 설정 (`/settings`)

**파일:** `client/src/pages/Settings.tsx`

### 구성 요소
- **글씨 크기**: 작게(18px) | 보통(20px) | 크게(24px)
- **음량**: 슬라이더 0-100% (기본 70%)
- **음성 속도**: 느림(0.75x) | 보통(0.9x) | 표준(1.0x) | 빠름(1.25x)
- **자동재생**: 체크박스 (기본 활성)
- **정보**: 앱 이름 + 버전

### 동작
1. 설정값 localStorage 저장
2. "저장 및 돌아가기" → 홈으로 이동
3. 설정 변경 즉시 반영

## 7. 공통 기능

### 음성 인식 (STT)
- Web Speech API 사용 (ko-KR)
- 마이크 권한 자동 요청
- 상태: 대기 → 녹음 → 처리 → 완료/오류
- iOS Safari 폴백 (텍스트 입력)

### 음성 합성 (TTS)
- Web Speech API 사용
- 한국어 음성, 0.9x 기본 속도
- 상태 메시지 자동 음성 출력

### PWA
- 홈 화면 설치 가능 (standalone 모드)
- Service Worker 캐싱 전략
- 오프라인 기본 기능 지원
- 앱 바로가기: 음성 검색, AI 추천, 내 서재

### 에러 처리
- ErrorBoundary 컴포넌트로 크래시 방지
- 404 페이지 (홈으로 안내)
- 네트워크 오류 시 재시도 안내
